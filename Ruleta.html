Descripción del Proyecto

Esta herramienta fue utiliza para evaluar oralmente a los alumnos de una forma más dinámica y llamativa, sin la necesidad de utilizar internet, usando la consola de comando y “php -S localhost:8000” para simular una página web. La versión origina utiliza php para eliminar los contenidos del bloc de notas para evitar la repetición de nombres o preguntas.

Este proyecto es una página web diseñada para evaluar de forma oral los contenidos de geografía en secundaria, haciéndolo más dinámico e interactivo. Su funcionamiento se divide en tres etapas: primero, se selecciona aleatoriamente un estudiante desde una lista almacenada en un bloc de notas (en la versión original con PHP, el nombre se eliminaba tras ser elegido). Luego, se gira una ruleta que puede otorgar ventajas o desafíos, como "vale doble", "ayuda de un amigo" o "eliminar una opción". Finalmente, se elige una pregunta al azar junto con sus opciones de respuesta desde otro bloc de notas, eliminándola después de ser utilizada. Esto permite realizar evaluaciones más equitativas y motivadoras, incentivando la participación de los alumnos.



Incluye:

  • Selector de nombres de un bloc de notas

  • Ruleta con ventajas aleatorias

  • Selector de preguntas y opciones de un bloc de notas
